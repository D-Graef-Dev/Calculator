# Calculator v0.1.0

Ein einfacher Taschenrechner in C#, der grundlegende mathematische Operationen durchführt.

## Features

- Addition (+)
- Subtraktion (-)
- Multiplikation (*)
- Division (/)
- Eingabevalidierung für Zahlen und Operatoren
- Fehlerbehandlung (z.B. Division durch Null)
- Endlosschleife für mehrere Berechnungen

## Wie man es nutzt

1. Starte das Programm
2. Gib die erste Zahl ein
3. Wähle einen Operator (+, -, *, /)
4. Gib die zweite Zahl ein
5. Das Ergebnis wird angezeigt
6. Du kannst weitere Berechnungen durchführen oder das Programm beenden

## Anforderungen

- .NET Framework oder .NET Core
- Visual Studio oder eine andere C# IDE
